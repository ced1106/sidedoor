Side Door Boston | Premium Nightlife Portal

sidedoor is a web application designed for a premier Boston nightlife promotion service. It bridges the gap between high-end aesthetic branding and functional lead generation by utilizing a custom-coded frontend with a Google Forms/Sheets backend.

The Mission
To provide an exclusive "Side Door" access point for Boston's club scene, offering seamless guest list registration and ticketed event discovery.


 Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Grid/Flexbox), JavaScript (ES6+)
- **Data Layer:** Google Sheets API (via CSV Publish)
- **Backend:** Google Forms (Headless Integration)
- **Hosting:** Netlify (with Continuous Deployment via GitHub)
